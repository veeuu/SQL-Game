1. **🚪 Gate of SELECT** (Rookie)

   - Column selection optimization

2. **⚔️ Joins of Doom** (Rookie)
   - INNER JOIN mastery
   - GROUP BY aggregation

3. **🌊 Subquery Swamp** (Pro)
   - IN vs EXISTS optimization
   - Subquery performance
   - JOIN alternatives
   - Query plan analysis

4. **📊 Aggregation Tower** (Pro)
   - GROUP BY with HAVING

   - Complex aggregations

5. **🪟 Window Cave** (Pro)
   - Window functions

6. **🔍 Index Labyrinth** (Optimizer)
   - Index awareness

7. **🔄 Recursive Depths** (Optimizer)

8. **🐉 Query Dragon** (Optimizer)

## 🎯 Mini Games

### 1. SQL Match 🎯
- **Type**: Memory card game
- **Objective**: Match SQL keywords with their functions
- **Points**: Up to 50 points
- **Skills**: Keyword recognition, SQL vocabulary
- **Mechanics**: 
  - Flip cards to find matching pairs
  - Keywords: SELECT, WHERE, JOIN, GROUP BY, etc.
  - Functions: Retrieve data, Filter rows, etc.
  - Score based on moves and time

### 2. Query Race ⚡
- **Type**: Speed typing challenge
- **Objective**: Type SQL queries as fast as possible
- **Points**: Up to 100 points
- **Skills**: Query syntax, typing speed, muscle memory
- **Mechanics**:
  - Display target query
  - Type it exactly
  - Progress through multiple queries
  - Time-based scoring

### 3. Index Puzzle 🧩
- **Type**: Multiple choice quiz
- **Objective**: Choose optimal indexing strategies
- **Points**: Up to 150 points
- **Skills**: Performance optimization, index selection
- **Mechanics**:
  - Given a query scenario
  - Choose best index strategy
  - Immediate feedback
  - Progressive difficulty

## 📊 Progress Tracking

### GitHub-Style Activity Heatmap
- **Visual**: Green cube grid (365 days)
- **Data**: Daily query completion count
- **Colors**: 
  - Empty: No activity
  - Light green: 1-2 queries
  - Medium green: 3-5 queries
  - Dark green: 6-9 queries
  - Brightest green: 10+ queries
- **Motivation**: Build streaks, stay consistent

### Statistics Dashboard
- **Current Level**: Progress through 8 levels
- **Total Score**: Cumulative points earned
- **Energy**: Attempt management (100 max)
- **Queries Solved**: Daily and total counts
- **Achievements**: Unlocked badges
