# ⚔️ SQL Dungeon

**An AI-powered SQL learning game with real-time multiplayer battles**

🌐 **Live Demo:** [https://veeuu.github.io/SQL-Game/](https://veeuu.github.io/SQL-Game/)

---

## What is SQL Dungeon?

SQL Dungeon turns learning SQL into a game. Progress through 15–100 levels of AI-generated SQL challenges, compete against friends in real-time duo battles, and earn points as you master everything from basic SELECT queries to recursive CTEs.

---

## Features

### Solo Mode
- Choose your challenge length at signup — 15, 30, 60, 100 days or any custom number
- Each level has a unique AI-generated challenge (no two are the same)
- Progressive difficulty: Beginner → Intermediate → Advanced
- Hints powered by Gemini AI
- Query validation by Gemini (accepts alternative correct solutions)
- Activity grid tracking your daily progress

### Duo Battle Mode
- Create a room and share the code with a friend
- Real-time split-screen gameplay via WebSocket
- Best of 3 rounds — different AI challenge each round
- First correct submission wins the round (double points)
- Live typing indicators and opponent status
- Match history on your dashboard

### Mini Games
- 🃏 **Memory Match** — match SQL keywords with their meanings
- ❓ **SQL Trivia** — timed multiple choice questions
- 🔨 **Query Builder** — drag words to build correct SQL queries

### Dashboard
- Live stats: level, score, completed levels, queries solved
- 30-day activity grid
- Recent duo match history (wins/losses)

## Local Development

### Prerequisites
- Python 3.11+
- Node.js 18+
- PostgreSQL

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt
cp .env.example .env         # fill in your credentials
python setup_postgres.py     # create tables
python main.py               # starts on http://localhost:8000
```

### Frontend
```bash
cd frontend
npm install
npm start                    # starts on http://localhost:3000
```

### Environment Variables (backend/.env)
```
PG_HOST=localhost
PG_PORT=5432
PG_DB=sql_dungeon
PG_USER=postgres
PG_PASS=your_password
GEMINI_API_KEY=your_gemini_key
GEMINI_MODEL=gemini-3.1-flash-lite-preview
```

## How It Works

### AI Challenge Generation
Every challenge is generated by Gemini AI based on:
- Current level (1–100)
- SQL concept being tested (SELECT → Recursive CTEs)
- Round number (ensures variety across rounds)

### Query Validation
Gemini validates queries by **logic and structure**, not exact match — so `SELECT * FROM students WHERE gpa > 3.5` and `SELECT id, name FROM students WHERE gpa > 3.5` are both accepted if the objective is met.

### Real-time Duo Mode
WebSocket connections keep both players in sync:
- Typing indicators
- Submission notifications
- Round results
- Match over broadcasts
