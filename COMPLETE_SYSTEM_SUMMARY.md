## 🤖 Gemini AI Features

### 1. Question Generation
Each round gets a unique challenge:
```python
generate_round_challenge(level=1, round_number=1)
# Returns different challenge than round 2 or 3
```

### 2. Hints
Progressive hints when stuck:
```python
get_hint_from_gemini(challenge, hint_number=1)
# Hint 1: General direction
# Hint 2: Specific keywords
# Hint 3: Almost the solution
```

### 3. Validation
Smart checking of queries:
